ðŸ”µ STEP 3: Create GitHub Action (CI/CD Pipeline)

Go to Actions tab

Click "Set up a workflow yourself"

You will see:

.github/workflows/main.yml


Paste this simple CI/CD workflow:
-----------------------------------------------------------------

name: Sample CI/CD

on:
  push:
    branches:
      - main

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - name: Checkout code
      uses: actions/checkout@v3

    - name: Display files
      run: ls -R

    - name: Run build script
      run: bash build.sh

    - name: Print build success
      run: echo "CI/CD pipeline completed!"

--------------------------------------------------------------
